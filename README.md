# Playwright Automation con Java 


# 1️⃣ Descripción del proyecto

Proyecto de automatización de pruebas end-to-end utilizando Playwright con Java.
Incluye estructura de testing organizada, configuración reutilizable y buenas prácticas de QA Automation.
Ideal para aprender y practicar automatización con Java y Playwright.


# 2️⃣ Objetivo

- Aplicar conceptos de automatización de pruebas.
- Practicar configuración mediante archivos properties.
- Construir una base reutilizable ("BaseTest").
- Mejorar habilidades y buenas prácticas en QA Automation.


# 3️⃣ Estructura del Proyecto

src
├── main
└── test
├── java
│   └── org.example
│       ├── test
│       │   └── EjemploTests.java
│       └── utils
│           ├── BaseTest.java
│           ├── Config.java
│           └── CustomOptions.java
└── resources
└── config.properties


# 4️⃣ Tecnologías Utilizadas

- Java
- Playwright
- Maven
- JUnit




# 5️⃣ Ejecución

1- Clonar el repositorio: git clone <URL-del-repositorio>
2- Configurar las dependencias con Maven: "mvn install"
3- Ejecutar las pruebas: "mvn test"
